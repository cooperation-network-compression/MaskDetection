基于YOLOv5增强模型的口罩佩戴检测

This is a face mask wearing detection system based on enhanced yolov5 model. The system provides two modes of image detection and real-time detection. The former can get the detailed information of the image by uploading the image, while the latter can realize remote real-time monitoring. When there is no mask, the system will automatically broadcast the warning until the target wears the mask.
